What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment. It is a method to frequently deliver apps to customers by introducing automation into the stages of app development. Continuous integration refers to an automation process for developers, while continuous deployment automates the delivery of the code to a production environment. This helps in finding bugs faster and deploying updates quickly.

My Pipeline

After I push my code to the main branch on GitHub, a GitHub Action is triggered. This action automatically builds my project, checks if the required files exist, and then deploys the updated version to GitHub Pages.
